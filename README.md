# 🛒 Online Retail II - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project presents a comprehensive **Exploratory Data Analysis (EDA)** of the **Online Retail II** dataset. The objective is to analyze customer purchasing behavior, identify data quality issues, explore sales trends, and generate actionable business insights that can support the development of a future recommendation system.

The analysis is performed using **Python** in **Jupyter Notebook**, with visualizations created using **Matplotlib** and **Seaborn**.

---

## 🎯 Objectives

- Understand customer purchase behavior.
- Explore the dataset and identify data quality issues.
- Analyze the top-selling products by quantity and revenue.
- Evaluate sales performance across different countries.
- Visualize monthly revenue trends.
- Detect outliers using box plots.
- Analyze relationships between numerical variables using a correlation heatmap.
- Generate business insights based on the analysis.

## ▶️ How to Run

```bash

```

### 2. Navigate to the project directory

```bash
cd Online-Retail-II-EDA
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

### 4. Download the dataset

Download the dataset from the **Google Drive** link provided above.

### 5. Place the dataset

Copy the downloaded file into the project directory.

### 6. Open the notebook

Open `Online_Retail_II_EDA.ipynb` using **Jupyter Notebook** or **Visual Studio Code**.

### 7. Run the notebook

Execute all cells from top to bottom.

## 💡 Key Insights

- The United Kingdom contributes the largest share of total revenue.
- A small number of products generate most of the company's sales.
- Revenue increases significantly during the holiday season.
- The dataset contains missing Customer IDs and duplicate records.
- Outliers exist due to cancelled transactions and bulk purchases.
